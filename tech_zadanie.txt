Техническое задание для проекта по Django "Социальная сеть":

Цель проекта:
Создать социальную сеть с возможностью регистрации пользователей, добавления друзей, создания и редактирования постов, работой с картинками, системой рекомендаций, интеграцией телеграм-бота, ролями, лайками, комментариями, документацией, избранным, пагинацией, личными сообщениями и отправкой оповещений почты с помощью Celery.

Функциональные требования:

1.    Регистрация и аутентификация пользователей с помощью электронной почты и пароля.
2.    Добавление и удаление друзей.
3.    Создание, редактирование и удаление постов с возможностью добавления картинок.
4.    Работа с системой рекомендаций на основе анализа интересов пользователей.
5.    Интеграция телеграм-бота для уведомления пользователей о новых постах и сообщениях.
6.    Разделение пользователей на роли с соответствующими ограничениями (администратор, модератор, пользователь).
7.    Лайки и комментарии к постам.
8.    Документация проекта, включая описание функций и API.
9.    Система избранного для сохранения понравившихся постов.
10.   Пагинация для удобства навигации по спискам постов.
11.   Личные сообщения между пользователями.
12.   Отправка оповещений почты с помощью Celery.
13.   Деплой проекта на сервер.

Нефункциональные требования:

    Использование Django в качестве основной платформы.
    Использование Bootstrap для создания пользовательского интерфейса.
    Разработка проекта в Git репозитории.
    Тестирование и отладка проекта перед выпуском.
    Размещение проекта на удаленном сервере с использованием Nginx и Gunicorn.    

Ожидаемый результат:
Работающая социальная сеть с функциональностью пользователей, друзей, постов, работой с картинками, системой рекомендаций, интеграцией телеграм-бота, ограничениями ролей, лайками, комментариями, документацией, избранным, пагинацией, личными сообщениями и отправкой оповещений почты с помощью Celery.
